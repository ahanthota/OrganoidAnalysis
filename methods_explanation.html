Basic Thresholding

Baseline

How it works: Calculates a global intensity value (Otsu's method) that best separates foreground pixels from background pixels.

‚úì Pros: Computationally instant. Robust for single, high-contrast organoids.

‚úó Cons: Fails completely when organoids touch, merging them into one large blob.

Watershed Segmentation

Recommended

How it works: Treats image intensity like a topographic map. It finds "peaks" (bright centers) and floods "valleys" (dark edges) to form basins. Where basins meet, it builds a dam (boundary).

‚úì Pros: Excellent at splitting touching objects. Mathematically robust.

‚ö† Note: Can over-segment (split one object into two) if texture is rough.

Hough Circle Transform

Volume Focused

How it works: Uses a voting system in parameter space. Edge pixels "vote" for potential circle centers. The algorithm detects objects even if they are partially occluded or overlapping.

‚úì Pros: The only method to detect 3D overlap (one behind another). Allows spherical volume calculation.

‚úó Cons: Assumes organoids are perfect circles.

StarDist (Geometric)

AI / Hybrid

How it works: Designed to mimic a Deep Learning model that predicts "star-convex" polygons. This implementation uses aggressive distance transforms to find geometric centers of round-ish objects.

‚úì Pros: Best-in-class for dense, crowded fields of view. Handles irregular but convex shapes better than Hough.

Advanced Morphology

Bio-Insights

How it works: Post-processing workflow inspired by MOrgAna/OSCAR. It extracts features like Convex Hull, Solidity, and fitted Ellipses to classify shape health.

‚úì Pros: Provides biological context (Is it dying? Is it budding?). Estimates cell counts.

‚Ñπ Use for: Drug screening or phenotypic analysis.
